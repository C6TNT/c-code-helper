# C 语言代码理解助手 V0.1 发布说明

## 产品定位

`C 语言代码理解助手` 是一个面向蓝桥杯单片机新生的代码讲解工具。

它不做通用 C 语言教学，也不做编译器，而是专门解决一个问题：

- 新生看见模板代码时，不知道每一行在干什么

## V0.1 这版能做什么

这版已经支持：

- 粘贴一段 C 代码
- 直接载入内置模板示例
- 识别常见语法元素
- 判断代码更像哪类赛题逻辑
- 给出赛题要求映射提示
- 解释常见术语
- 提示“如果要改这段代码，先看哪里”
- 复制讲解结果

## 当前适合怎么用

这版最适合这几种场景：

1. 新生第一次看模板代码，看不懂变量、函数、结构体在干什么
2. 看见一段页面、按键、参数或报警逻辑，不知道它在解题里对应哪部分要求
3. 想先知道“这段代码该从哪开始看、从哪开始改”

## 当前版本判断

`V0.1` 已经是可运行、可演示、可小范围试用的版本。

但它现在更适合：

- 小范围试用
- 收集新生反馈
- 继续小步迭代

而不是直接当成完全成熟的正式版。
