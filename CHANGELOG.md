# 更新日志

## V0.1

首个可运行试用版，当前已经具备这些能力：

- 粘贴一段 C 代码进行讲解
- 内置模板示例代码，一键载入体验
- 识别常见语法元素
- 判断代码更像哪类赛题逻辑
- 给出赛题要求映射提示
- 给出术语解释
- 给出“如果要改这段代码，先看哪里”的提示
- 支持复制讲解结果
- 已完成首版 Windows `exe` 打包验证

### V0.1 后续迭代补充

- 增加“这段代码具体做了哪些动作”卡片，不再只按大类场景做泛化讲解
- 识别函数调用、被修改变量、EEPROM 写入、按键取值、页面切换等更具体的代码动作
- 让 `App_SaveParam`、`App_HandleKey`、显示函数、采样函数的右侧讲解差异更明显
- 补充“真正改了什么变量”和“关键调用是什么”，减少不同代码讲解看起来都一样的问题
