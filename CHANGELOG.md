# 更新日志

## V0.4

这一版开始让产品更像“模板联动导航器”，重点补上了：

- 新增“这段代码和模板里哪几个函数最相关”卡片
- 不再只说这段代码属于哪类逻辑，而是继续告诉用户应该顺着哪些函数往下看
- 针对页面显示、按键处理、参数保存/读取、采样、报警这几类逻辑，增加了更具体的模板函数联动提示
- 右侧讲解区进一步和真实模板工程阅读路径对齐
- 重打 Windows `exe`

## V0.3

这一版开始从“能讲代码”走向“开始真正帮助改题”：

- 新增“这段代码依赖谁”卡片
- 新增“改这里会影响哪里”卡片
- 新增“代码执行链”卡片
- 增强“如果你要改这段代码，先看哪里”的提示
- 强化高风险改动提醒
- 统一整理核心分析层和界面文案
- 修正一批中文乱码文本
- 重打 Windows `exe`

## V0.2

针对真实客户反馈“右侧讲解太宽泛、不同代码看起来讲得都差不多”做了一轮重点迭代：

- 增加“这段代码具体做了哪些动作”卡片
- 新增对关键调用的识别，例如：
  - `AT24C02_WriteByte / ReadByte`
  - `SEG_SetCode / SEG_SetDigit`
  - `Key_GetEvent`
  - `DS18B20 / DS1302 / PCF8591`
  - 频率、超声波、串口相关调用
- 增加对“真正被修改的变量”的识别
- 让参数保存、按键处理、显示函数、采样函数的讲解差异更明显
- 优化“如果你要改这段代码，先看哪里”的提示
- 重打 Windows `exe`

## V0.1

首个可运行试用版，具备这些能力：

- 粘贴一段 C 代码进行讲解
- 内置模板示例代码，一键载入体验
- 识别常见 C 语法元素
- 判断代码更像哪类赛题逻辑
- 给出赛题要求映射提示
- 给出术语解释
- 给出“如果要改这段代码，先看哪里”的提示
- 支持复制讲解结果
- 完成首版 Windows `exe` 打包验证
