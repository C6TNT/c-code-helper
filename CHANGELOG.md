# 更新日志

## V0.2

针对真实客户反馈“右侧讲解太宽泛、不同代码看起来讲得都差不多”做了一轮重点迭代：

- 增加“这段代码具体做了哪些动作”卡片
- 新增对关键调用的识别，比如：
  - `AT24C02_WriteByte / ReadByte`
  - `SEG_SetCode / SEG_SetDigit`
  - `Key_GetEvent`
  - 页面切换变量
  - 参数保存 / 参数读取
  - 温度 / ADC / 频率 / 距离 / RTC 相关动作
- 新增对“被修改变量”的识别，让讲解更贴近代码真正改了什么
- 让 `App_SaveParam`、`App_HandleKey`、显示函数、采样函数的讲解差异更明显
- 优化“如果你要改这段代码，先看哪里”的提示，不再只按大类泛讲
- 重打 Windows `exe`，可直接给新生和客户继续试用

## V0.1

首个可运行试用版，具备这些能力：

- 粘贴一段 C 代码进行讲解
- 内置模板示例代码，一键载入体验
- 识别常见 C 语法元素
- 判断代码更像哪类赛题逻辑
- 给出赛题要求映射提示
- 给出术语解释
- 给出“如果要改这段代码，先看哪里”的提示
- 支持复制讲解结果
- 完成首版 Windows `exe` 打包验证
