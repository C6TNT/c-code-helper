# C语言代码理解助手

`c-code-helper` 是一个面向蓝桥杯单片机新生的 C 代码理解助手。

当前版本：
- `V0.4`

它不是通用 C 语言教学软件，也不是编译器，而是专门解决这种真实问题：
- 新生看到模板代码时，不知道每一行在干什么
- 不知道这段代码更像对应赛题的哪一类要求
- 不知道如果要改代码，应该先盯住哪些变量、函数和接口
- 不知道改一个地方后，还会连带影响哪里
- 不知道这段代码和模板工程里的哪些函数最相关

## V0.4 重点变化

这一版开始从“能讲代码”继续升级到“能带着顺藤摸瓜看模板”：

- 新增 `这段代码和模板里哪几个函数最相关`
  - 会结合当前代码里的调用、语义标签和模板典型结构
  - 直接告诉你应该顺着哪些函数名字继续往下看
- 让页面显示、按键处理、参数保存/读取、采样、报警这几类代码的联动提示更不一样
- 右侧讲解区进一步靠近真实模板阅读路径

## 当前能力

- 粘贴一段 C 代码
- 直接载入内置模板示例
- 识别常见 C 语法元素
- 用中文解释这段代码大概在做什么
- 判断这段代码更像哪类赛题逻辑
- 给出“对应哪类赛题要求”的映射
- 解释常见术语
- 识别“这段代码具体做了哪些动作”
- 识别“这段代码依赖谁”
- 提示“改这里会影响哪里”
- 给出代码执行链
- 给出“这段代码和模板里哪几个函数最相关”
- 支持复制讲解结果
- 已有可运行 Windows `exe`

## 适合谁用

- 大一新生
- C 语法基础一般
- 能打开工程，但看到模板代码容易发怵
- 想尽快进入“能看懂、能改题”的状态

## 运行方式

先安装依赖：

```powershell
pip install -r requirements.txt
```

再启动：

```powershell
python app/main.py
```

也可以直接双击：

`run_app.bat`

或者使用已经打包好的：

`dist/C语言代码理解助手.exe`
