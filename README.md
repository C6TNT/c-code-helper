# C语言代码理解助手

`c-code-helper` 是一个面向蓝桥杯单片机新生的 C 代码理解助手。

它不是通用 C 语言教学软件，也不是编译器，而是专门解决这种真实问题：

- 新生看见模板代码时，不知道这段代码到底在干什么
- 不知道这段代码更像对应赛题里的哪一类要求
- 不知道如果要改这段代码，应该先看哪些变量、哪些函数、哪些联动点
- 不知道改完一处后，还要同步检查哪里

当前版本：

- `V0.6`

## V0.6 重点变化

这一版开始继续往“顺着模板往下看”推进：

- 新增 `这段代码更可能在哪些 .c/.h 文件里成套出现`
- 现在不只提示相关函数，还会提示你下一步该去翻 App、BSP、Devices 还是 Drivers 哪类文件
- 页面显示、按键处理、参数保存/读取、采样、报警、串口等逻辑，都补了更明确的文件联动提示

## V0.5 重点变化

上一版开始明显往“改题辅助器”方向走：

- 新增 `这段代码常和哪些变量一起联动`
- 新增 `改完这里后还要同步检查哪几处`
- 新增 `这段代码大概会在哪里被调用`
- 保留并继续强化：
  - 赛题要求映射
  - 具体动作识别
  - 依赖识别
  - 模板相关函数联动提示
  - 影响范围提示

## 当前能力

- 粘贴一段 C 代码进行讲解
- 一键载入内置模板示例
- 识别常见 C 语法元素
- 判断这段代码更像哪类赛题逻辑
- 说明这段代码更像对应哪类赛题要求
- 解释常见术语
- 识别这段代码具体做了哪些动作
- 识别这段代码依赖谁
- 提示这段代码和模板里哪几个函数最相关
- 提示这段代码常和哪些变量一起联动
- 提示改这里会影响哪里
- 提示改完这里后还要同步检查哪几处
- 提示这段代码大概会在哪里被调用
- 提示这段代码更可能在哪些 `.c/.h` 文件里成套出现
- 给出代码执行链
- 给出建议阅读顺序
- 支持复制完整讲解结果
- 已有可运行 Windows `exe`

## 适合谁用

- 大一新生
- C 语法基础一般
- 能打开工程，但看到模板代码容易发怵
- 想尽快进入“能看懂、能改题”的状态

## 运行方式

先安装依赖：

```powershell
pip install -r requirements.txt
```

再启动：

```powershell
python app/main.py
```

也可以直接双击：

- `run_app.bat`

或者使用已打包好的：

- `dist/C语言代码理解助手.exe`
