# C语言代码理解助手

`c-code-helper` 是一个面向蓝桥杯单片机新生的 C 代码理解助手。

当前版本：
- `V0.2`

它不是通用 C 语言教学软件，也不是编译器，而是专门解决一个实际问题：

- 新生看到模板代码时，不知道每一行在干什么
- 不知道这段代码对应赛题的哪一部分要求
- 也不知道如果要改，应该先改哪里

## V0.2 重点变化

这一版重点解决了真实客户反馈：

> 右侧讲解比较宽泛，不同代码看起来讲得都差不多。

所以 V0.2 不再只按“页面 / 按键 / 参数”这种大类去解释，而是开始识别这段代码里更具体的动作，例如：

- EEPROM 写入 / 读取
- 按键取值
- 页面切换
- 数码管显示
- 参数保存 / 参数读取
- 温度 / ADC / 频率 / 距离 / RTC 相关动作
- 哪些变量真的被修改了

这样 `App_SaveParam`、`App_HandleKey`、显示函数、采样函数的讲解会明显更不一样。

## 当前能力

- 粘贴一段 C 代码
- 直接载入内置模板示例
- 识别常见语法元素
- 用中文解释代码大概在做什么
- 判断这段代码更像哪类赛题逻辑
- 给出“对应哪类赛题要求”的映射
- 解释常见术语
- 告诉你“如果要改这段代码，先看哪里”
- 识别“这段代码具体做了哪些动作”
- 支持复制讲解结果
- 已有可运行 Windows `exe`

## 适合谁用

- 大一新生
- C 语法基础一般
- 能打开工程，但看到模板代码容易发怵
- 想尽快进入“能看懂、能改题”的状态

## 运行方式

先安装依赖：

```powershell
pip install -r requirements.txt
```

再启动：

```powershell
python app/main.py
```

也可以直接双击：

`run_app.bat`

或使用已经打包好的：

`dist/C语言代码理解助手.exe`
