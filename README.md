# C语言代码理解助手

`c-code-helper` 是一个面向蓝桥杯单片机新生的 C 代码理解助手。

当前版本：
- `V0.3`

它不是通用 C 语言教学软件，也不是编译器，而是专门解决这类真实问题：

- 新生看到模板代码时，不知道每一行在干什么
- 不知道这段代码更像对应赛题的哪一类要求
- 不知道如果要改代码，应该先盯住哪些变量、函数和接口
- 不知道改一个地方后，还会连带影响哪里

## V0.3 重点变化

这一版开始从“能讲代码”升级到“开始帮助改题”：

- 新增 `这段代码依赖谁`
  - 直接告诉你这段代码依赖哪些变量
  - 依赖哪些函数调用
  - 依赖哪些外设接口
- 新增 `改这里会影响哪里`
  - 提醒你改参数、页面、显示、报警、采样这类代码时，最容易漏同步的地方
- 新增 `代码执行链`
  - 把一段代码拆成“输入 -> 中间处理 -> 最后输出”
  - 让 `App_SaveParam`、`App_HandleKey`、`App_ShowDataPage` 这类代码讲出来更不一样
- 增强 `如果你要改这段代码，先看哪里`
  - 增加高风险改动提醒
  - 更贴近真实改题场景

## 当前能力

- 粘贴一段 C 代码
- 直接载入内置模板示例
- 识别常见 C 语法元素
- 用中文解释这段代码大概在做什么
- 判断这段代码更像哪类赛题逻辑
- 给出“对应哪类赛题要求”的映射
- 解释常见术语
- 识别“这段代码具体做了哪些动作”
- 识别“这段代码依赖谁”
- 提示“改这里会影响哪里”
- 给出代码执行链
- 支持复制讲解结果
- 已有可运行 Windows `exe`

## 适合谁用

- 大一新生
- C 语法基础一般
- 能打开工程，但看到模板代码容易发怵
- 想尽快进入“能看懂、能改题”的状态

## 运行方式

先安装依赖：

```powershell
pip install -r requirements.txt
```

再启动：

```powershell
python app/main.py
```

也可以直接双击：

`run_app.bat`

或者使用已经打包好的：

`dist/C语言代码理解助手.exe`
