# C语言代码理解助手

`c-code-helper` 是一个面向蓝桥杯单片机新生的 **模板代码理解与改题辅助工具**。

它不想做成“大而全的 C 语言教材”，而是专门解决这些真实问题：

- 新生看到模板代码，不知道这段代码到底在干什么
- 不知道这段代码更像赛题里的哪一类要求
- 不知道这段代码依赖谁、改这里会影响哪里
- 不知道改完一处后，还要同步检查哪些变量、函数和文件
- 想用 AI 继续深讲，但又不想一上来就只靠大模型胡猜

当前版本：
- `V0.7`

## V0.7 重点变化

这一版开始把产品推进到“本地规则 + 可选 AI 深入讲解”的结构：

- 新增 `AI 深入讲解` 区域
- 新增应用内 `AI 设置`
- 新增 `测试连接` 按钮
- 未配置 `API Key` 时自动进入 `AI 预览模式`
- AI 结果会被整理成 3 张更像学长讲解的卡片：
  - `这段代码更像在干什么`
  - `你应该先盯住哪几个点`
  - `如果你准备改它，最容易漏掉哪里`
- 支持复制完整 AI 讲解、复制 AI 摘要、复制单张 AI 卡片

## 当前能力

### 本地规则讲解

- 粘贴一段 C 代码进行讲解
- 一键载入内置模板示例
- 识别常见 C 语法元素
- 判断这段代码更像哪类赛题逻辑
- 说明这段代码更像对应哪类赛题要求
- 解释常见术语
- 识别这段代码具体做了哪些动作
- 识别这段代码依赖谁
- 提示这段代码和模板里哪几个函数最相关
- 提示这段代码常和哪些变量一起联动
- 提示改这里会影响哪里
- 提示改完这里后还要同步检查哪几处
- 提示这段代码大概会在哪里被调用
- 提示这段代码更可能在哪些 `.c/.h` 文件里成套出现
- 给出代码执行链
- 给出建议阅读顺序

### AI 深入讲解

- 支持应用内配置 `API Key / Base URL / Model`
- 支持连接测试
- 支持 AI 预览模式
- 支持 AI 深入讲解
- 支持复制 AI 完整结果、摘要、单张卡片

## 适合谁用

- 大一新生
- C 语法基础一般或偏弱
- 能打开模板工程，但看到代码会发怵
- 想尽快进入“能看懂、能顺着改题”的状态

## 运行方式

先安装依赖：

```powershell
pip install -r requirements.txt
```

再启动：

```powershell
python app/main.py
```

也可以直接双击：

- `run_app.bat`

或者使用已打包好的：

- `dist/C语言代码理解助手.exe`
