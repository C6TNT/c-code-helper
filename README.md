# C 语言代码理解助手

`c-code-helper` 是一个面向蓝桥杯单片机新生的 C 代码理解助手。

当前版本：

- `V0.1`

它不是通用 C 语言教学软件，也不是编译器，而是专门解决一个问题：

- 新手看见模板代码时，不知道每一行在干什么

V0.1 先聚焦最小能力：

- 粘贴一段 C 代码
- 直接载入内置模板示例
- 识别常见语法元素
- 用中文解释代码大概在做什么
- 判断这段代码更像哪类赛题逻辑
- 给出赛题要求映射提示
- 解释常见术语
- 告诉你“如果要改这段代码，先看哪里”
- 识别这段代码“具体做了哪些动作”，比如按键取值、EEPROM 写入、页面切换、显示输出
- 给出推荐阅读顺序

## 运行方式

先安装依赖：

```powershell
pip install -r requirements.txt
```

再启动：

```powershell
python app/main.py
```

也可以双击：

`run_app.bat`

## V0.1 目标

第一版优先保证：

1. 能正常粘贴代码
2. 能直接载入内置模板示例
3. 能输出可读的中文讲解

6. 能把代码和赛题要求、常见术语、修改入口联系起来
